# GoCar Rental System

A full-stack Car Rental Management System developed as an MCA project using Node.js, Express.js, EJS and MySQL.

## Features

- User Registration & Login
- Admin Login
- Browse Premium Cars
- Car Booking System
- Booking Confirmation Email
- PDF Booking Receipt
- Admin Dashboard
- Booking Search
- Booking Status Management
- Responsive Design

## Technologies Used

- Node.js
- Express.js
- EJS
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
- Nodemailer
- PDFKit

## Installation

```bash
git clone https://github.com/Yashraj8420/GoCar-Rental-System.git

cd GoCar-Rental-System

npm install

npm start
```

## Environment Variables

Create a `.env` file.

```
DB_HOST=localhost
DB_USER=your_mysql_username
DB_PASSWORD=your_mysql_password
DB_NAME=gocar

EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password
```
